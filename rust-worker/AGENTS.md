## CONVENTIONS

### Async Architecture
- `#[tokio::main]` - multi-threaded runtime
- `spawn_blocking` for WASM execution (non-blocking)
- `Arc<Mutex<T>>` for shared state (`running`, `num_running`)
- `mpsc::channel` for result passing

### Memory
- `#[global_allocator] static ALLOC: mimalloc::MiMalloc` - performance allocator
- `StoreLimitsBuilder` for WASM memory bounds

### WASM Execution
- Each task: new `Store`, `Module`, `Linker`, `Instance`
- Fuel: `store.set_fuel()` + `consume_fuel(true)`
- JSON args → WASM Val conversion in `run_wasm`

### Semaphore Concurrency
- `Arc<Semaphore>` limits parallel WASM executions
- Acquire permit before `spawn_blocking`
